# Customer-churn-EDA
Project Overview

Customer churn is a major challenge for subscription-based businesses. Understanding why customers leave can help companies design more effective retention strategies.

This project performs an Exploratory Data Analysis (EDA) on a customer churn dataset to identify behavioral and demographic patterns associated with customer attrition.

The analysis focuses on getting insights related to customer demographics, spending behavior, subscription types, and payment behavior.

# Objectives

The goal of this project is to:

Explore the structure and quality of the dataset

Identify patterns associated with customer churn

Analyze relationships between customer attributes and churn

Generate business insights that could support retention strategies

# Tools

The analysis was conducted using:

Python

Pandas for data manipulation

Matplotlib for visualization

Seaborn for statistical visualizations

Jupyter Notebook
